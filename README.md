# Kunuz
This project aims to build Arabic language resources (e.g. corpora and gold standards) for various Information Retrieval (IR) and text mining tasks including knowledge extraction and document classification. 

We provide the following resources:
<br/>- <a href='https://github.com/bounhasibrahim/Kunuz/tree/Kunuz-IR'>Kunuz-IR</a>: a bilingual Information Retrieval (IR) collection built according to <a href="https://trec.nist.gov/">TREC</a> guidelines and data formats.
